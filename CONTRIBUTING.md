# Contributing to Snake-Water-Gun Game

Thank you for your interest in contributing! Here's how you can help:

## Getting Started

1. **Fork the repository**
2. **Clone your fork**: `git clone https://github.com/YOUR_USERNAME/snake-water-gun-game.git`
3. **Create a branch**: `git checkout -b feature/your-feature`
4. **Make your changes**
5. **Commit**: `git commit -m 'Add clear, descriptive message'`
6. **Push**: `git push origin feature/your-feature`
7. **Open a Pull Request**

## Code Standards

- Follow **PEP 8** style guidelines
- Add **docstrings** to all functions
- Include **error handling** (try-except blocks)
- Write **meaningful variable names**
- Add **comments** for complex logic
- Test all features before submitting

## Pull Request Guidelines

- Provide a clear description of changes
- Reference any related issues
- Include test results or gameplay screenshots
- Keep commits atomic and focused

## Feature Suggestions

Have ideas for improvements? Open an issue with:
- Feature description
- Use cases
- Proposed implementation

## Report Issues

Found a bug? Open an issue with:
- Clear title
- Detailed description
- Steps to reproduce
- Expected vs actual behavior

## Questions?

Feel free to open a discussion or issue!

Thanks for contributing! 🚀
