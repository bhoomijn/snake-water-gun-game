# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it by emailing **bhoomijn4@gmail.com** instead of using the issue tracker.

### What to Include

Please include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Any suggested fixes (if available)

## Response Timeline

- **Acknowledgment**: Within 24 hours
- **Initial Assessment**: Within 2-3 business days
- **Resolution**: Depends on severity

## Supported Versions

| Version | Status | Support Until |
|---------|--------|---------------|
| 1.0.x   | Active | 2027-07-01    |

## Security Best Practices

When using this project:
- Keep dependencies updated
- Review code before using in production
- Report any suspicious activity
- Follow PEP 8 and security guidelines

Thank you for helping keep this project secure! 🔐
